# e-portfolio

Site statique d'un e-portfolio avec cinq pages :

- "index.html" — Accueil
- "activity.html" — page activité 
- "career-development.html" — page carrière 
- "engineering-course.html" — page études 
- "civic-engagement.html" — page engagement civil
- "contact.html" — page pour contacter 
- "mobility.html" — mobilité  

Fichiers principaux : `style.css` pour le style commun.

Pour voir localement : ouvrir `index.html` dans un navigateur ou lancer un serveur statique, par exemple :

# e-portfolio
